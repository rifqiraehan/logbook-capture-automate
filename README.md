## Otomatisasi Screenshot Tabel Logbook

Otomatisasi Screenshot Tabel Logbook pada semua minggu kegiatan untuk menjaga ukuran tetap konsisten dengan auto-crop bagian tabel.

## Cara Menggunakan

1. Buka Halaman Logbook di Menu `Entry Logbook KP`
2. Tekan `Ctrl + Shift + I`
3. Masuk ke tab Console.
4. Tempel seluruh kode `console.js`.
5. Tekan Enter.
6. Browser akan mulai mengunduh screenshot per minggu secara otomatis. Pastikan untuk memberi perizinan untuk mengunduh multiple files.
